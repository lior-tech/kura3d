# KURA3D — Douro Valley

Static render-slideshow site for **KURA**, a five-star spa resort in the Douro Valley, Portugal, by BOA Hotels.
Live at **https://kura3d.boahotels.com** (GitHub Pages).

- `index.html` — the whole site (no build step, self-contained)
- `assets/` — fonts (PP Fragment Glare), KURA + BOA logos, favicons, share image
- `img/web/` — the ten 2026 renders, web-optimised

Deploy: push to `main`; GitHub Pages serves the repo root. Custom domain set via `CNAME`.
